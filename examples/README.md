# Examples

Worked configurations for common ClawFirm deployments. Each subdirectory has a README + the seed file + the relevant overlays.

- `solo-laptop/`        — single-user prosumer install on a 16 GiB MacBook Pro.
- `smb-vps/`            — 1-3 node SMB cluster on a typical VPS provider with a single 24 GiB GPU.
- `enterprise-airgap/`  — air-gapped Enterprise install with FIPS-mode build.
